Pitchfork contracts and their explanatory reader are maintained in the
Pitchfork repository. Exploratory architecture, roadmap, embodied-AI, research,
and client-incubation inputs are maintained outside this public documentation
tree.

## Working Rule

Pancakes owns product identity, node architecture, ecosystem governance, and
public documentation. Pitchfork owns reusable contracts, accounting,
settlement, resource-state, projection, and database-independent persistence
semantics.
